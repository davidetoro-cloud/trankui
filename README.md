# Trankui

Piattaforma operativa per trovare professionisti audiovisivi disponibili e costruire crew affidabili.

## Architettura

- Frontend statico HTML, CSS e JavaScript
- Autenticazione, database, policy RLS e realtime su Supabase
- Hosting previsto tramite GitHub Pages con dominio `trankui.com`

La chiave Supabase presente in `config.js` e una chiave pubblicabile per browser. Le credenziali amministrative e la password del database non sono incluse nel repository.
